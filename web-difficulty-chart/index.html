Web制作 難易度チャート（松竹梅）

作りたい機能を選ぶと、だいたいの難易度と想定コストの目安がわかります（目安です。案件により変動します）。

梅 — 手軽
静的ページ、プロフィール紹介、簡単なランディングなど。ノーコードやテンプレートで無料〜低コストから始めやすいゾーンです。
  • テキスト中心・画像少なめ
  • 更新は年に数回でよい
竹 — ひと工夫
フォームや定期更新、埋め込みなど。レンタルサーバーやSaaSの月額を見込むことが多いです。
  • 軽い動的処理・外部埋め込み
  • デザイン・アニメの調整
松 — 本格
ログイン・決済・リアルタイム連携など。開発者の工数か、継続的なインフラ・API費用が前提になりやすいです。
  • データベース・認証
  • 外部サービス連携・AI API
