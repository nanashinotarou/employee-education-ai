# 実装計画: Day 10「アプリケーション開発の基礎知識」構築

## 1. 概要
本タスクは、「コース情報共有」Day 10 と `Today_Research.md` の内容に基づき、初心者向けに最適化されたチュートリアルページ **`vol10-1.html`** を新規作成するものです。
Day 8・9 の「Web制作（静的サイト）」からテーマを「Web開発（動的アプリ）」にピボットする回であり、受講生が「**Web制作とWeb開発の違い**」「**フレームワーク**」といったアレルギーを起こしやすい概念を、専門用語を排除した比較・図解で直感的に掴めるよう実装します。あわせて `index.html` の Day10 カードの遷移先も更新します。

## 2. ユーザーレビュー事項
> [!IMPORTANT]
> - 本計画は実装を **Cursor** など他のエディタ／AI に引き継ぐ前提で構築しています。Cursor が迷わないよう、コピペ可能な本文原稿・使用クラス名・CSS変数・素材ファイルパスを明示しています。
> - 素材フォルダに **Bolt.new / GitHub Copilot / Dify / Padlet / GAS のアイコン画像は存在しません**（2026-04-22 時点）。Cursor 側では Font Awesome の代替アイコン（`fa-bolt` `fa-github` `fa-robot` `fa-note-sticky` `fa-file-code` 等）を使用する方針とします。GitHub Copilot は既存 `素材/GitHubアイコン画像.png` で代替可。
> - YouTube動画タイトルは `get_titles.py` / `fetch_pl.js` で取得できます。取得が困難な場合は本計画書に記載の仮タイトルで固定。

### UI・コンポーネント設計方針
- **親テーマの踏襲**: `vol09-1.html` の `:root` CSS変数・クラス定義（`.glass-card` / `.bento-item` / `.step-box` / `.pub-card` / `.info-box` / `.highlight-box` / `.compare-grid` / `.prompt-box` / `.alert-warn` / `.video-card` / `.task-item` / `.wax-seal` / `.day-badge` / `.tab-btn` / `.tab-content` / `.mission-area` / `.inline-icon` / `.heading-icon`）をそのまま流用。**新規CSSクラスの追加は原則禁止**（追加する場合はユーザーに確認）。
- **動画組み込み**: `GEMINI.md` §1 に従い、すべてのYouTube動画は **Facade パターン**で実装。`.vc-thumb[data-video-id]` + `maxresdefault.jpg` / `hqdefault.jpg` フォールバック背景 + Vanilla JS の `once:true` クリックハンドラで `<iframe>` に差し替える。
- **レスポンシブ**: `GEMINI.md` §2 の `@media (max-width: 768px)` テンプレートを `<style>` 末尾に必ず含める。
- **配色**: 桜ピンク（`--accent`）をベースに維持。Day10 は「アプリ開発へのピボット回」であることを示すアクセントとして、概念解説ブロックに限り `--tier-advanced`（青系）・`--purple` を差し色として使用（桜系を上書きしない）。
